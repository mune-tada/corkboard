# Corkboard - VSCode コルクボード拡張機能

Scrivener風のコルクボード。ワークスペース内のファイルを「カード（付箋）」として表示し、ドラッグ&ドロップで並べ替え・整理できるVSCode拡張機能。

## 機能

- **カード表示**: Markdown/テキストファイルをカード（付箋）として表示
- **グリッドモード**: カードを格子状に配置、ドラッグで並べ替え
- **フリーフォームモード**: カードを自由な位置に配置（Scrivenerのフリーフォームコルクボード風）
- **ラベル**: 色分けラベルでカードを分類
- **ステータス**: 「未着手」「下書き」「推敲中」「完成」等のステータス管理
- **概要表示**: ファイルの先頭行やfrontmatterから自動取得、手動編集も可能
- **ファイル連携**: カードをクリックするとファイルが開く

## 使い方

1. コマンドパレット(`Cmd+Shift+P`) → 「コルクボードを開く」
2. 「＋ ファイルを追加」ボタンでファイルを選択
3. カードのヘッダーをドラッグして並べ替え
4. カードの `⋯` ボタンからラベル・ステータス設定、概要編集

## 開発

```bash
npm install          # 依存関係インストール
npm run build        # ビルド
npm run watch        # ウォッチモード
```

F5キーでExtension Development Hostが起動します。

## ローカルインストール

```bash
npm run package      # .vsixファイル生成
code --install-extension corkboard-0.1.0.vsix
```
